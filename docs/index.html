    M²·Alpha 是什么

    M²-Alpha (Micro-Macro Alpha Engine) 是一个自研的深度学习 A 股 截面 alpha 选股模型，配套一套完全公开的执行策略，并通过 GitHub Actions 每日 自动更新数据。模型权重、策略规则、回测代码、数据 pipeline 全部开源在 GitHub。

    区别于一次性回测展示，本项目的实证记录通过 daily live tracking 持续累积 —— 每过一个交易日，OOS 记录在 git history 中新增一条，删不掉、改不了，谁都能审计。

    交易策略 · 6 条规则

    1. 候选池仅在沪深 300 成分股内选股
    2. 推荐数每日 Top 10（从评分前 30 候选筛出）
    3. 权重等权 10% / 只，不按评分加权
    4. 行业上限同一申万一级行业 ≤ 2 只
    5. 卖出规则掉出当日评分 Top 50 才卖（滞后带）
    6. 成交09:30 集合竞价，双边 0.13% 手续费

    如何参考使用 · 3 步

    1. 每晚 18:30 后来本站查看次日推荐 （数据自动更新）
    2. 次日 09:25 在券商软件按 10 只清单录入"市价委托"， 09:30 集合竞价撮合。每只配净值 10%
    3. 第三日 09:30 开盘卖出，回笼资金。再来本站看新清单

    数据更新机制

    每个交易日北京时间 18:30，GitHub Actions cron 自动执行：

    1. BaoStock 拉取沪深 300 当日量价 + 基本面
    2. 加载 ml/m2alpha.pt 计算今日截面评分
    3. 增量更新 docs/data/data.json
    4. commit + push → GitHub Pages 自动重新部署

    周末与节假日自动跳过（数据没更新就不 commit）。哪天跑挂大概率是 BaoStock 字段微调，到 GitHub Action 里看日志。

    风险提示 · 务必阅读

    • 不构成任何形式的投资建议 — 本站为模型预测的公开展示
    • 过去的回测不代表未来 — 模型有效性可能随市场结构衰减
    • 历史回测中曾出现单日 -9.21% 的最差日（2025-10-09），跟单需有承受短期回撤的心理准备
    • 严格按规则执行 — 只买几只 / 不按权重 / 隔几天才换 — 实际效果会与回测显著不同
    • 建议至少配置 10 万以上本金参与，否则手续费占比过高吞噬收益
    • 据此操作的任何决策，由使用者自行承担风险
